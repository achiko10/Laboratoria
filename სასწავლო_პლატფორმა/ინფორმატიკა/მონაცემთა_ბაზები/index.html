სქემა:

📊 შედეგი

SQL მოთხოვნის შედეგი აქ გამოჩნდება
SQL ბრძანებები:
SELECT — მონაცემების წაკითხვა
INSERT INTO — ახალი ჩანაწერის დამატება
UPDATE — არსებული მონაცემის შეცვლა
DELETE — ჩანაწერის წაშლა
WHERE — პირობა, ფილტრი
ORDER BY — დალაგება (ASC/DESC)
← მთავარზე დაბრუნება
